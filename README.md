# Ansible betacloud.bootstrap

This is an Ansible role that bootstraps an Ubuntu servers.

Supported Linux distributions
-----------------------------

* Ubuntu 16.04 (Xenial)
* CentOS 7

Dependencies
------------

none

Example Playbook
----------------

```yml
- hosts: all

  roles:
  - role: betacloud.bootstrap
```
